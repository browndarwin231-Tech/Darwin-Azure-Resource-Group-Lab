# Darwin-Azure-Resource-Group-Lab
Help Desk and Azure lab demonstrating cloud administration, resource management, and Azure fundamentals through the creation and management of an Azure Resource Group.

---

## Objectives

- Access the Microsoft Azure portal
- Navigate to Azure Resource Groups
- Create and verify an Azure Resource Group
- Review Resource Group details
- Explore Azure Tags
- Review the Azure Activity Log

---

## Technologies Used

- Microsoft Azure
- Azure Portal
- Azure Resource Manager
- Azure Activity Log

---

## Skills Demonstrated

- Azure Portal Navigation
- Cloud Resource Management
- Resource Group Administration
- Azure Tagging
- Activity Log Review
- Cloud Monitoring Basics
- Help Desk / SOC Documentation

---

## Resource Details

| Setting | Value |
|---|---|
| Resource Group Name | Darwin-RG-Lab01 |
| Subscription | Azure subscription 1 |
| Region | East US |
| Deployment Method | Azure Portal |

---

## Steps Performed

1. Signed in to the Microsoft Azure portal.
2. Opened the Azure Home Dashboard.
3. Navigated to Resource Groups.
4. Created the Resource Group named `Darwin-RG-Lab01`.
5. Verified the Resource Group was created successfully.
6. Opened the Resource Group Overview page.
7. Reviewed the Tags section.
8. Reviewed the Activity Log for resource activity.

## Key Takeaways

- Azure Resource Groups help organize cloud resources.
- Resource Groups are tied to a subscription and region.
- Tags help categorize and manage resources.
- Activity Logs help track administrative actions and resource changes.
- Azure portal skills are useful for Help Desk, Cloud Support, and SOC Analyst roles.
